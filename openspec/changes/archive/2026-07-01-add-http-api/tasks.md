> Built with the travel project's API conventions. Depends on `add-message-bus` (command dispatch)
> and accounts, transfers, idempotency, projections. New deps: `baldinof/roadrunner-bundle` +
> `symfony/runtime` (app server), `nyholm/psr7` + `symfony/psr-http-message-bridge`,
> `symfony/security-bundle` (firewall), `symfony/validator` + `symfony/serializer`. OpenAPI is
> generated by a custom reflection generator (no nelmio/swagger-php) and verified without an
> OpenAPI-validator dependency. The production image and Helm are deferred to add-deployment.

## 1. HTTP stack & RoadRunner

- [x] 1.1 Enable the Symfony HTTP stack: `public/index.php` (runtime front controller); framework HTTP config (validation + serializer); `config/routes/api.yaml` importing attribute routes from `src/Api` under `/api`; load `App\Api\` Actions tagged `controller.service_arguments` in `services.yaml` (Request/Response DTOs excluded).
- [x] 1.2 Add RoadRunner: `baldinof/roadrunner-bundle` + `symfony/runtime` (`APP_RUNTIME`), `public/index.php` returning the Kernel; `.rr.yaml` + `.rr.dev.yaml`; the `rr` binary + `ext-sockets` in the dev image; a compose `http` service running `rr serve -c .rr.dev.yaml`.

## 2. Cross-cutting concerns

- [x] 2.1 `ApiKeyAuthenticator` on the Symfony Security firewall: validate the API-key header; `access_control` requires it on `/api/*`; `health`/`openapi` are public. Missing/invalid → `401`.
- [x] 2.2 Idempotency listener over mutating routes: `begin` (hash request), replay Completed, `409` InProgress, `422` Mismatch; capture the response and `complete`.
- [x] 2.3 `ApiExceptionListener` (`KernelEvents::EXCEPTION`) mapping throwables via `match()` to status and rendering RFC 9457 `application/problem+json` (`type`/`title`/`status`/`detail`, plus an `errors` member for validation). Explicit cases per the design table (ledger exceptions extend `\RuntimeException`).
- [x] 2.4 `Request` DTOs + `symfony/validator` constraints (`#[MapRequestPayload]`); validation failure → `422` with field details via the error listener.
